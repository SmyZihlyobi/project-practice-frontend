# Фронтенд по проектной деятельности

## Для конфигурации env

```
NEXT_PUBLIC_BACKEND_URL=<URL на котором держится бек>
```

## Запуск локальной версии

### Установка пакетов
```
yarn
```
### Локальный билд 

```
yarn dev
```

## Запуск прод версии

```
docker build -t pp-front .
docker run -p 3000:3000 pp-front
```
